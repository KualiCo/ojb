The 1.x release of OJB will be a big step forward in usability and integration in existing frameworks (like Spring).

08/2005 - Linguine Maps supports OJB

Linguine Maps is an open-source (LGPL, Java 1.4+) utility that will automatically produce easy to read UML-style entity-relation diagrams from OJB mapping files.
